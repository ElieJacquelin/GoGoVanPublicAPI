GoGoVan API documentation

Getting a quote

Request:

curl -X GET \
  -H 'GoGoVan-API-Key: 90a723b8-cdaa-43f2-93fb-c3ff48697937' \
  -H 'GoGoVan-User-Language: zh-CN' \
  -F 'order[city]=beijing' \
  -F 'order[name]=dawei' \
  -F 'order[phone_number]=14716021413' \
  -F 'order[pickup_time]=2016-03-21T17:24:01H' \
  -F 'order[vehicle]=sbread' \
  -F 'order[extra_requirements][remark]=woshidaweila' \
  -F 'order[remark]=Please confirm the following 3 items: http://example.com/list' \
  -F 'order[extra_requirements][need_carry]=true' \
  -F 'order[extra_requirements][passenger_count]=4' \
  -F 'order[locations]=[[39.940066,116.461059,"北京市朝阳区三里屯路19号院-10号楼"],[39.907648,116.406453,"北京市东城区东交民巷44号"]]' \
  'http://gogovan-dev-cn.herokuapp.com/api/v0/orders/price.json'

GoGoVan-User-Language can be en-US zh-TW zh-CN

Response:

{
    "breakdown":{
    "fee":{
    "title":"车费",
    "value":50}
    },
    
    "distance_in_kms":"8.381",
    "base":50
    }

http://example.com/list' \

Canceling a request

curl -v -X POST \
-H 'GoGoVan-API-Key: 5ccaee19-f269-423c-ae6b-5382b079c92f' \
-H 'GoGoVan-User-Language: zh-CN' \
'https://gogovan-dev-cn.herokuapp.com/api/v0/orders/4823/cancel.json'

Response:

The response will be just HTTP 200 OK with no response body.

Order status

Pending - The order is not picked up by any driver yet
Active - A driver has been assigned to the order
completed - The order has been marked "completed" by the driver
canelled - The order has been marked "cancelled" by the driver

Mandatory fields

  GoGoVan-API-Key
  GoGoVan-User-Language
  order[city]
  order[name]
  order[phone_number]
  order[pickup_time]
  order[vehicle]
  order[locations]
  //The location of start point should be within the [city] 
  API URL
